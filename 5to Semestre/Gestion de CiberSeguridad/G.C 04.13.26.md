# Normas, Gobernanza y Herramientas de Gestión

## 1. Gobernanza en Ciberseguridad

### Definición
La gobernanza en ciberseguridad es el conjunto de estructuras, políticas y procesos que aseguran que la seguridad de la información:

- esté alineada con los objetivos del negocio
- gestione riesgos adecuadamente
- cumpla regulaciones

---

## 2. Objetivos de la Gobernanza

| Objetivo               | Descripción                   |     |
| ---------------------- | ----------------------------- | --- |
| Alineación estratégica | Seguridad alineada al negocio |     |
| Gestión de riesgos     | Identificación y control      |     |
| Cumplimiento           | Normativas y leyes            |     |
| Valor                  | Protección de activos         |     |

---

## 3. Principios Clave

- responsabilidad clara
- transparencia
- control
- mejora continua

---

## 4. Normas y Estándares

## ISO 27001

### Definición
Estándar internacional para gestión de seguridad de la información (SGSI).

---

### Componentes

| Elemento | Función |
|----------|--------|
| Políticas | reglas de seguridad |
| Controles | medidas de protección |
| Auditorías | evaluación |
| Mejora continua | optimización |

---

## NIST Cybersecurity Framework

### Funciones

| Función | Descripción |
|--------|------------|
| Identify | identificar activos |
| Protect | proteger sistemas |
| Detect | detectar amenazas |
| Respond | responder incidentes |
| Recover | recuperar operaciones |

---

## COBIT

### Definición
Marco de gobernanza y gestión de TI.

---

### Objetivo
- control de procesos TI
- alineación con negocio

---

## ITIL

### Definición
Framework de gestión de servicios TI.

---

### Enfoque
- entrega de valor
- mejora continua

---

## 5. Comparación de Estándares

| Estándar | Enfoque | Uso |
|----------|--------|-----|
| ISO 27001 | seguridad | SGSI |
| NIST | ciberseguridad | gestión de riesgos |
| COBIT | gobernanza TI | control |
| ITIL | servicios TI | operación |

## 9. Buenas Prácticas

- aplicar estándares internacionales

- actualización de sistemas

---

## 10. Relación Gobernanza + Herramientas

- gobernanza define qué hacer
- herramientas ejecutan cómo hacerlo

---

## 11. Problemas Comunes

| Problema | Causa |
|----------|------|
| falta de control | sin gobernanza |
| ataques exitosos | mala seguridad |
| incumplimiento | falta de normas |
| baja visibilidad | sin herramientas |

---

## 12. Idea Clave

La ciberseguridad efectiva se basa en:

- normas → estructura
- gobernanza → control
- herramientas → ejecución

Sin integración de estos tres elementos, la seguridad es incompleta.
