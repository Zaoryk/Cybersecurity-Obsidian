# REDES, CLOUD, CIBERSEGURIDAD, GESTIÓN Y TALENTO

## 1. Redes de Datos (Base de Todo)

### Concepto
- Conjunto de dispositivos interconectados que intercambian información.

### Componentes
- Dispositivos finales (PC, móvil)
- Dispositivos de red (router, switch)
- Medios (cable, WiFi)

### Características clave
- Tolerancia a fallos → sigue funcionando ante errores
- Escalabilidad → crece sin afectar rendimiento
- QoS → prioriza tráfico crítico
- Seguridad → protege datos

### Tipos de red
- PAN → personal (Bluetooth)
- LAN → local (oficina)
- MAN → ciudad
- WAN → global (Internet)

## 4. Ciberseguridad

### Conceptos
- Amenaza → posible ataque
- Vulnerabilidad → debilidad
- Riesgo → probabilidad de daño

---

### Tipos de ataques
- Malware (ransomware, troyanos)
- Phishing (engaño)
- DDoS (saturación)
- MitM (intercepción)
- APT (ataques avanzados)

---

### Vulnerabilidades comunes
- malas configuraciones
- software desactualizado
- errores de acceso
- inyecciones (SQL, XSS)

---

### Seguridad clave
